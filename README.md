# Thermodynamic-Cycles-Simulator-
Streamlit App for Thermodynamic Cycle Simulation  This app simulates Otto, Diesel, and Dual combustion cycles using both air-standard and fuel-air models. It calculates state points, heat transfer, work, efficiency, and generates P-V and T-S diagrams, along with PDF export. Built with Python, NumPy, SciPy, and Streamlit.
